# Classification using decision tree-based models to predict the risk of stroke.

Classification project using decision tree-based models to predict the risk of stroke based on patient health data, including age, gender, medical history, and lifestyle factors. The project includes data preprocessing, model training, and evaluation of classification performance.

The sprawozdanie_DD.pdf file contains description and results of code from dd.ipynb file (in Polish).

## Tools
- Python
- sklearn
- numpy
- pandas
- matplotlib
